
# 🧠 Smart Productivity Assistant

A smart AI-powered productivity assistant built using Google Gemini AI.

## 🎯 Chosen Vertical
Productivity — Smart Task & Calendar Assistant

## 💡 Approach and Logic
- User types a task in plain English
- Google Gemini AI analyzes the task and suggests priority and time
- Tasks are displayed in a clean, accessible UI
- Users can add and delete tasks dynamically

## ⚙️ How the Solution Works
1. User enters a task description
2. App sends the task to Google Gemini API
3. Gemini returns smart suggestions about the task
4. Task is added to the list with AI feedback shown

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- Google Gemini AI API

## 📌 Assumptions Made
- User has a modern browser
- Gemini API key needs to be replaced for live use
